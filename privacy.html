Privacy Policy

Data Collection and Use

Hatenna is a Discord bot built for Pokétwo-related utility features, including predictions, collections, shiny hunts, feeds, leaderboards, daycare tools, AFK systems, and server settings. Hatenna is intended to respect user privacy as much as possible and only stores or processes data that is necessary for the bot to function.

  • Discord user IDs, guild IDs, channel IDs, role IDs, and message IDs
  • Server and channel configuration data
  • Pokétwo-related data such as Pokemon names, forms, IDs, timestamps, catches, hatches, unboxes, balances, and related gameplay metadata
  • Feature-specific state such as collections, shiny hunts, reserve data, AFK timers, daycare data, feed data, leaderboard data, and related settings
  • Global sync data, including OAuth authorization state and synced server information where applicable

Message Processing

Hatenna does not read or store ordinary user chat messages except where necessary to process commands, command replies, supported Pokétwo-related messages, or other feature-specific interactions used by the bot.

  • Commands explicitly run by users
  • Replied-to or referenced messages where a feature depends on that interaction
  • Supported Pokétwo messages, embeds, attachments, and related metadata required for bot functionality

Hatenna does not monitor or archive general user conversation for unrelated purposes.

Purpose of Processing

Any stored or processed data is used only for the operation of Hatenna’s intended functionality. This includes prediction and ping features, collections, shiny hunts, daycare tools, feeds, leaderboards, settings, OAuth-backed global sync, service stability, abuse prevention, and technical diagnosis.

Hatenna does not sell user data. Stored data is not shared with third parties except where necessary for service operation, such as hosting, database, caching, logging, or related infrastructure services under controlled access.

User Control

Where supported, users may control how their information is displayed or used in public-facing systems. This may include opting out of certain feed or leaderboard visibility, hiding identifying information where anonymity controls are supported, and disabling, clearing, or unsyncing certain feature states. Supported tracking or display systems may also be managed through available bot commands, including h!toggle where applicable.

Data Retention

Data is retained only for as long as reasonably necessary to support bot functionality, operational integrity, and feature continuity. Data may be removed, cleared, or become inactive when a server removes the bot, a feature is reset, cleared, disabled, or unsynced, a user opts out of a supported feature, stale data is cleaned up, or a valid removal request is made where applicable and technically feasible.

Third Parties

Hatenna operates on Discord and may process Pokétwo-related content only as required for supported bot functionality. If a user uses OAuth-backed global sync, authorization is handled through Discord’s OAuth system. Hatenna may rely on third-party infrastructure providers for hosting, storage, caching, logging, or similar technical services necessary to operate the bot. Hatenna is not responsible for the privacy practices of Discord, Pokétwo, or any other third-party platform.

Security

Reasonable efforts are made to minimize collected data and restrict access to operational systems. However, no service can guarantee absolute security.

Changes

This Privacy Policy may be updated from time to time to reflect changes in Hatenna’s functionality, infrastructure, or privacy practices. Continued use of Hatenna after such changes constitutes acceptance of the revised policy.
